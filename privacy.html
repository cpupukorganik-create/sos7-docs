🛡️ Kebijakan Privasi

SOS7 TikTok Uploader — Versi 1.0, 26 Juni 2026

Kami di SOS7 Garrison menghargai privasi Anda. Kebijakan privasi ini menjelaskan bagaimana aplikasi SOS7 TikTok Uploader mengumpulkan, menggunakan, dan melindungi informasi Anda saat Anda menggunakan layanan kami.

1. Informasi yang Kami Kumpulkan

Aplikasi ini mengumpulkan informasi berikut untuk keperluan operasional:

  • Data Produk: Nama produk, harga, toko, dan link afiliasi yang diambil dari database internal kami.
  • Data Video: File video yang akan diunggah ke TikTok.
  • Data Autentikasi: Token akses OAuth untuk berkomunikasi dengan API TikTok (disimpan secara aman di server).
✅ Kami TIDAK mengumpulkan:
  • Informasi pribadi pengguna TikTok (nama, email, nomor telepon, lokasi).
  • Data pengguna lain di TikTok.
  • Informasi finansial, kartu kredit, atau data pembayaran.
  • Data dari perangkat Anda di luar konteks aplikasi.

2. Bagaimana Kami Menggunakan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Mengunggah video ke TikTok sesuai permintaan Anda.
  • Mengelola konten afiliasi yang dipromosikan melalui video.
  • Meningkatkan kinerja dan efisiensi aplikasi (misalnya, rotasi produk).
  • Menganalisis performa video (views, engagement) melalui API TikTok.

3. Perlindungan Data

Kami menerapkan langkah-langkah keamanan yang wajar untuk melindungi data Anda:

  • Enkripsi: Token akses disimpan dalam file terenkripsi di server.
  • Koneksi Aman: Semua komunikasi dengan TikTok menggunakan HTTPS.
  • Akses Terbatas: Hanya sistem yang berwenang yang dapat mengakses data.
  • Penyimpanan Lokal: Data produk dan video hanya disimpan di server internal, tidak di cloud publik.

4. Berbagi Data dengan Pihak Ketiga

Kami tidak membagikan data Anda dengan pihak ketiga, kecuali:

  • Dipersyaratkan oleh hukum atau peraturan yang berlaku.
  • Dengan persetujuan eksplisit dari Anda.

Catatan: Video yang diunggah ke TikTok akan diproses oleh TikTok sesuai dengan kebijakan privasi mereka. Kami tidak mengontrol pengumpulan data oleh TikTok.

5. Penyimpanan Data

Data produk dan log aktivitas disimpan di server kami selama diperlukan untuk operasional aplikasi. Anda dapat meminta penghapusan data Anda kapan saja.

6. Hak Pengguna

Anda memiliki hak untuk:

  • Mengakses data yang kami simpan tentang Anda.
  • Memperbaiki data yang tidak akurat.
  • Menghapus data Anda dari sistem kami.
  • Menarik persetujuan atas penggunaan data kapan saja.

7. Cookie dan Pelacakan

Aplikasi ini tidak menggunakan cookie atau teknologi pelacakan lainnya. Kami hanya menggunakan token akses yang diperlukan untuk autentikasi API.

8. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diumumkan melalui halaman ini dengan tanggal revisi terbaru. Kami menyarankan Anda untuk meninjau halaman ini secara berkala.
